Vehicle System (Open Source)

Credits:
This vehicle system was built using @Krokosha666's suspension system (YouTube) as the original base/inspiration.
Huge thanks and full credit to them for the foundation of the suspension setup.

Features:
1. Auto Loader (includes automatic wheel/thruster setup and auto weld support)
2. Straightforward code that is easy to read, edit, and expand (Atleast for me...)
3. Simple vehicle configuration through: Vehicle_System.Configuration
4. Designed to be a flexible base for building your own custom vehicle system
5. Supports adding or duplicating as many wheels as needed

Notes:
- You can enable visible suspension raycasts in the "Control" LocalScript inside the "Injections" folder.
  Look for the variable: DebugSprings

- Creating new vehicles should be mostly plug-and-play.
  In most cases, you only need to place your vehicle model inside the correct "Body" (or corresponding) folders,
  then configure it properly in Vehicle_System.Configuration.

- Wheels can be moved freely or duplicated by copying the Thruster parts.
  No manual welding should be required if the Auto Loader is set up correctly.
